# 📚 Trabalhos de Java - Programação Orientada a Objetos (POO)

Bem-vindo ao meu repositório de trabalhos em Java desenvolvidos na disciplina de Programação Orientada a Objetos (POO). Aqui você encontrará atividades que aplicam conceitos fundamentais da POO, como classes, objetos, herança, polimorfismo, encapsulamento e interfaces.

# 🔹 Conteúdo do Repositório

Cada pasta representa uma atividade da disciplina. Alguns tópicos abordados incluem:

Classes e Objetos: Criação e manipulação de objetos em Java.
Herança: Relações entre classes pai e classes filhas.
Polimorfismo: Sobrescrita de métodos e comportamento dinâmico.
Encapsulamento: Uso de modificadores de acesso (private, public, protected) e métodos getters/setters.
Interfaces e Abstração: Implementação de interfaces e classes abstratas para modelagem de sistemas.
